✅ Design Decisions ที่ตกลงแล้ว

  • Font: Sarabun ทั้งระบบ (มีหัว ดูทางการ)
  • Theme: Light only
  • Desktop: 3-column layout (Modules · Sub-nav · Content)
  • Col 1 ย่อ icon-only เมื่อมี sub-nav
  • Mobile: Hamburger drawer max-width 85vw, word-break
  • List: Kebab menu (⋮) ทุก row สำหรับ actions
  • Transport ไม่ lock — scroll แก้ทุก step ได้ตลอด
  • Sub-form แทน modal: Consent / V/S / IV / Refer
  • Timeline = thin sticky bar ทั้ง 3 step
  • Mobile bottom action: 💾 บันทึก (สีเขียว) + ✓ Auto-saved indicator
  • Navigate back: ← เคสทั้งหมด ทุกหน้า form
  • Print, Share Public, Edit Patient, Delete = kebab ทุกที่
  • Monitor: คง V1 layout (มี function เยอะ)
  • Real V1 fields: ไม่เพิ่ม ไม่ลด ไม่เปลี่ยน function
  • iOS Print compatibility — webkit-print-color-adjust + page-break-inside + PWA fallback ด้วย html2pdf
  • Consent: ใช้ V1 ทั้งดุ้น (transport/consent-prototype.html)

⚠ iOS Print — ต้อง test ก่อน rollout

  • Background colors — `-webkit-print-color-adjust: exact`
  • PWA standalone — fallback ใช้ html2pdf.js
  • Page break — `page-break-inside: avoid` ทุก section
  • Hide UI chrome — topbar/sub-nav/FAB/kebab ซ่อนใน @media print
  • Font Sarabun embed — preload + fallback Tahoma
  • A4 size — `@page { size:A4; margin:12mm; }`
  • Test matrix: iOS Safari + iOS PWA + Android Chrome + Desktop
