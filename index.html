Modules ทั้งหมด
Transport
บันทึกนำส่งผู้ป่วย, V/S (ต้นทาง/ระหว่างทาง/ปลายทาง), IV, Consent + DNR (ยืนยัน 2 ขั้น + บัตร ปชช./พยาน), พิมพ์ใบส่งตัว, อัปโหลด PDF + OCR
First Aid
อีเวนท์ปฐมพยาบาล ลงทะเบียนผู้บาดเจ็บ, triage 4 ระดับ, V/S, บันทึกการรักษา, Quick Dispense ยา, รายงานสรุป
GPS Tracking
ตามรถ real-time, แชร์ลิงก์สาธารณะ (expire ได้), ดูกล้อง, หลาย Provider + ค้นหาสถานที่ใกล้สุด (โรงพยาบาล/คลินิก/ปั๊ม/ตำรวจ) + เส้นทาง/ETA + เตือนเร็วเกิน 120 km/h
Location
ฐานข้อมูลตำแหน่ง (โรงพยาบาล/ลูกค้า), แผนที่ + marker cluster, mini-map ซ้อนใน modal, แชร์ลิงก์สาธารณะ (admin ลบได้แม้ expired)
Monitor
Dashboard รวมสถานะ — เคส Active, อีเวนท์, GPS, กระพริบเตือนผู้ป่วยวิกฤต + ค้นหาสถานที่ใกล้สุดในแผนที่รถ
Admin Panel
จัดการตั้งค่า, Provider GPS, Map Provider (Leaflet/Google), เช็ค API, สถิติ, Activity Log, ภาพรวมระบบ
Architecture & Stack
Frontend
  • Bootstrap 5.3.3 + Sarabun Font
  • Leaflet 1.9.4 + MarkerCluster (แผนที่ + OSM)
  • Leaflet GoogleMutant (Google Tiles ใน Leaflet)
  • PDF.js 2.16.105 (อ่าน PDF)
  • Chart.js 4 (กราฟสถิติ)
  • SweetAlert2 (popup)
Backend & Services
  • Supabase (PostgreSQL + RLS + Realtime)
  • Cloudinary (รูป + PDF)
  • Google Apps Script (Auth HR)
  • CMSV6 / 808gps (GPS API)
  • Google Maps Platform (Maps JS / Places New / Directions / Distance Matrix / Geocoding) — เปิดตามการตั้งค่า
  • Cloudflare Worker (OCR / Proxy)
ฟีเจอร์เด่น (Phase 1-4)
  • ค้นหาสถานที่ใกล้สุด — โรงพยาบาล / คลินิก / ร้านยา / ปั๊ม / ตำรวจ / ดับเพลิง (Places API New)
  • นำทางจริงบนถนน + ETA (Directions API)
  • จัดอันดับตามเวลาจริง (Distance Matrix) — badge "⚡ เร็วสุด"
  • เตือนรถเร็วเกิน 120 km/h (border แดง + pulse)
  • Share Link (GPS / Location) + admin ลบ token หมดอายุได้
  • DNR workflow + ภาพบัตร ปชช./Passport/ใบราชการ + ลายเซ็นพยาน
  • Mini-map modal (Location) — ดูแผนที่ในหน้าเดียว
  • Fallback อัตโนมัติ — ไม่มี Google key → ใช้ Leaflet + ซ่อนฟีเจอร์ที่ต้องใช้ Google

Session & Security

ตั้งค่า session timeout และ force logout เมื่อสงสัยว่ามี security incident

Auto logout timeout 8 ชั่วโมง
User ทุกคนจะถูก logout อัตโนมัติเมื่อครบ 8 ชั่วโมงหลัง login
ตั้งค่านี้เป็น hard-coded ใน SESSION_TIMEOUT_MS (shared/auth.js + index.html)
Force Logout All Users

บังคับให้ทุก user login ใหม่ทันที (ภายใน 5 นาที) — ใช้เมื่อ:

  • สงสัยว่ามี account ถูก compromise
  • user ลืม logout แล้วเครื่องหาย
  • ต้องการ reset session ทั้งระบบ
